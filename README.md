# 🎲 Die Roll Simulation

## 📌 Overview
This project simulates rolling a six-sided die 1000 times using a random number generator and analyzes the resulting probability distribution.

## 🎯 Objectives
- Simulate a real-world random process
- Compare experimental results with theoretical probability
- Understand frequency distribution

## ⚙️ Technologies
- Python

## 🧠 How It Works
- Generates random numbers between 0 and 1
- Maps values to die faces (1–6)
- Counts occurrences
- Calculates percentage frequency for each outcome

## ▶️ How to Run
```bash
python simulation.py
